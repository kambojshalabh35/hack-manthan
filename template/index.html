About Us

The Sentiment Analysis Model integrated with our website. Which is empowered by a Django Backend Modelling Supervised Machine Learning Algorithm. Talking about the user interface, we have a website with complete prediction system installed into it. Type in how you feel and we will try to define whether its positive or negative

Sentiment Analysis - Opinion Mining.

Sentiment analysis relates to the use of natural language processing (NLP), text analysis and computational linguistics to identify and extract subjective information in text document.

Sentiment analysis strives to determine the attitude of a writer with respect to the topic or the overall contextual polarity of a text document. We can distinguish four general approaches to sentiment analysis: keyword spotting, lexical affinity, statistical methods, and concept-level techniques.

Our API service combines all aforementioned techniques to determine document sentiment, depending on many factors such as: document type (Twitter content, Amazon comment, email content etc.), it's length and lexical coherence
